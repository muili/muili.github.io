## login GitHub

GitHubのアカウントを取得。
[GitHub](https://github.com/)


## fork Jekyll

JekyllをForkする。リンク先の画面右上のForkボタンを押す。
[Jekyll Now repository](https://github.com/barryclark/jekyll-now)

Forkしたjekyll-nowリポジトリを開く。

Settingを押し、リポジトリ名を変更する。

before:
 jekyll-now

after:
 {{page.author}}.github.io

## ssh-keygen

SSH鍵を作成し、githubに登録しておく。
[github help generating-ssh-keys](https://help.github.com/articles/generating-ssh-keys/)


## git-clone

git clone git@github.com:{{page.author}}/{{page.author}}.github.io

上記のようにせず、取得先をhttps経由にすると毎度パスワードが聞かれる。
その解決策。

git remote -v
origin https://github.com/{{page.author}}/{{page.author}}.github.io (fetch)
origin https://github.com/{{page.author}}/{{page.author}}.github.io (push)

git remote set-url origin git@github.com:{{page.author}}/{{page.author}}.github.io

## AngularJS and Jekyll/Liquid

[Liquid manual](https://github.com/Shopify/liquid/wiki/Liquid-for-Designers)

テンプレートでAngularJSを有効にしておいて、AngularJSのsyntaxをJekyll/Liquidの(raw/endraw)に書く。

{% raw %}
<p ng-repeat="x in [1,2,3,4]">{{ x }}
{% endraw %}
